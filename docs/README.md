## Описание проекта «Симулятор раздачи листовок»

«Симулятор раздачи листовок» — это ироничный инди-проект в жанре симулятора повседневности, выполненный в уникальном визуальном стиле. Игроку предстоит примерить на себя роль промоутера, выживающего в нуарных городских декорациях и взаимодействующего с эксцентричными прохожими через систему нелинейных диалогов.

**Технологии:** C#, Unity, Git.
