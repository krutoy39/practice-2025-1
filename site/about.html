О проекте

«Симулятор раздачи листовок» — это ироничный инди-проект в жанре симулятора повседневности, выполненный в уникальном визуальном стиле.

🎯 Цель игры

Игроку предстоит примерить на себя роль промоутера, выживающего в нуарных городских декорациях. Нужно раздать определённое количество листовок за отведённое время.

⚙️ Основные механики

  • Взаимодействие с прохожими через систему нелинейных диалогов
  • У каждого персонажа свой характер и настроение
  • Разные реакции на предложение взять листовку
  • Ограниченное время на выполнение задания

🛠 Технологии

  • Язык программирования: C#
  • Игровой движок: Unity
  • Контроль версий: Git
